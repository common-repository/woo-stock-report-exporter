== Description ==

The WC Stock Report Exporter is a WooCommerce plugin that helps you expport store reports. This plugin downloaded the stock reports in a CSV (Comma-Separated Values) format, which is compatible with popular spreadsheet softwares.

== Installation ==

Step 1) Navigate to the Plugins=>Add New and search "WC Stock Report Exporter" in search text box.

Step 2) Click on install button and after installation click on Activate button.
